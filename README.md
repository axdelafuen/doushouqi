# Getting started

Clone the repository :

```shell
git clone https://codefirst.iut.uca.fr/git/axel.de_la_fuente/DouShouQi.git
```

Build dependencies :

```shell
cd ./Sources/Model
swift build
cd ../CommandLineExt
swift build
```

Run the app :

```shell
cd ../CommandLineTest
swift run
```
